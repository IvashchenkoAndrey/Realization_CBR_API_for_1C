Получение курса валют по API ЦБ на выбранный день. Доступен график на неделю или месяц. Конфигурация имеет всего одну обработку, собрана и протестирована на 1С:Предприятие 8.3 (8.3.27.1859). 
К сожалению запрос готовой динамики (https://cbr.ru/scripts/XML_dynamic.asp?date_req1=01/01/2026&date_req2=05/01/2026&VAL_NM_RQ=R01235) на актуальную дату API не отдает, 
вместо этого реализован запрос котировок на каждый день и перенесен в график.

Знаю что запрос в цикле это плохо, но иначе динамику на актуальную дату с API ЦБ не получить. Вот сам запрос (https://cbr.ru/scripts/XML_daily.asp?date_req=01/01/2026).

Построение графика на 30 дней может длится 5-6 секунд, это из-за ограничений API - 5 запросов в секунду.
